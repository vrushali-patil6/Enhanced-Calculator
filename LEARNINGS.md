## Challenges Faced

### Package Structure

Initially, the project was created without packages. After converting it into a Maven project, package-related errors occurred because the classes were located in different packages.

### Solution

I reorganized the project structure, added the correct package declarations and import statements, and verified that all classes communicated correctly.

---

## Hardest Bug

The most difficult issue was resolving package and import errors after restructuring the project. Some classes could not recognize each other because they belonged to different packages.

### Fix

I corrected the package declarations, imported the required classes properly, and organized the project into a clean Maven structure.
